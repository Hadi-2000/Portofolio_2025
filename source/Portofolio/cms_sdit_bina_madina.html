Objektif

Project kutawarkan ke sekolah untuk membuat website profil sekolah.

Alat

CMS Sekolahku, html, css, php, javascript, dan bootstrap.

Project ini dibuat untuk kebutuhan penyampaian informasi mengenai profil sekolah lewat website. Project ini memiliki 3 class pengguna yaitu pengunjung, guru dan admin.

Pada class pengunjung, pengunjung bisa melihat profil mengenai sekolah, dan melakukan pendaftaran

Pada class guru, guru bisa melakukan login, mengecek siswa yang baru mendaftar atau yang lama, menambah informasi tertentu dan mengubah profil guru.

Pada Class admin, admin bisa melakukan perubahan total ketampilan website, mengatur jadwal pemeliharaan, mengelola akun, mengubah dan menambah informasi.
